# Tutorial Website

A GeeksforGeeks-style tutorial website built with Next.js and TypeScript.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Tutorial categories (Java, Python, JavaScript, etc.)
- Individual tutorial pages
- Responsive design
- Search functionality
- Clean, professional UI

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
